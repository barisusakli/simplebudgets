### Development

Copy config.default.js to config.js, update db connection string if necessary.

Start mongodb

`npm run server` to start the backend

`npm run dev` to bundle client side resources

Access at http://127.0.0.1:3000

### Deploy

`git pull`

`npm run build`

`pm2 restart server`
